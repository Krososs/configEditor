# configEditor

1. Clone repository
2. Install Kivy
```commandline
pip install Kivy
```
3. Run
 ```commandline
python ConfigEditorApp.py
```
## Requirements
1. MongoDB Community Edition - https://www.mongodb.com/docs/manual/administration/install-community/
```commandline
pip install pymongo
```

2. Rebrickable API Key - https://rebrickable.com/
```commandline
pip install rebrick
```
